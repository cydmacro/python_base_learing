## 二、实操题(60分)

**实操题1(20分): JSON数据解析**
编写程序处理Label Studio导出的JSON:
```python
data = {
    "data": {"image": "/upload/cat001.jpg"},
    "annotations": [{
        "result": [{
            "value": {"choices": ["猫"]}
        }]
    }]
}
```
1. 提取文件名
2. 提取标签
3. 保存为CSV格式

**实操题2(40分): 标注质量检查**
编写程序:
1. 创建两个标注员的标注数据(各10条)
2. 计算标注一致率
3. 找出不一致的样本
4. 生成质检报告:
   - 总样本数
   - 一致样本数
   - 一致率
   - 不一致样本列表
